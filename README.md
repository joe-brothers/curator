## Environment
- pnpm
- next.js
- husky
- eslint
- prettier
- picocss

## Dev
```bash
npm i -g pnpm
```
1. install pnpm

```bash
pnpm install
```
2. install dependencies (also install husky)

```bash
pnpm dev
```
3. run dev server

4. Open [http://localhost:3000](http://localhost:3000)
